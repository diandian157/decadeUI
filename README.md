## 扩展介绍

无名杀游戏的界面美化扩展

## 使用方法

### 安装

1. 下载扩展文件
2. 将整个文件夹放入无名杀的 `extension` 目录下
3. 打开本体设置-选项-通用-自动导入扩展
4. 重启游戏，在扩展列表中启用"十周年UI"
