# 新增/修改样式指南

本文档记录在十周年UI中新增或修改一个样式需要改动的所有位置。

---

## 新增一个样式（以 threedol 为例）

### 1. 定义样式选项

**文件**: `src/config/definitions/appearance.js`

在 `newDecadeStyle.item` 中添加选项：
```js
item: {
    on: "十周年",
    off: "移动版",
    othersOff: "一将成名",
    onlineUI: "online",
    babysha: "欢乐三国杀",
    codename: "名将杀",
    threedol: "ol3d",    // ← 新增
},
```

### 2. 注册样式索引

**文件**: `src/core/decadeModule.js`

**2a.** 加入 `STYLE_OPTIONS` 数组（决定加载哪个 player CSS）：
```js
const STYLE_OPTIONS = ["on", "off", "othersOff", "onlineUI", "babysha", "codename", "threedol"];
//                      ↑player1  ↑player2  ↑player3      ↑player4     ↑player5   ↑player6   ↑player7
```
> `STYLE_OPTIONS.indexOf("threedol") === 6` → 加载 `player7.css`

**2b.** 加入 `STYLE_TO_SKIN` 映射（决定 UI 皮肤文件）：
```js
const STYLE_TO_SKIN = {
    on: "shizhounian",
    off: "shousha",
    othersOff: "xinsha",
    onlineUI: "online",
    babysha: "baby",
    codename: "codename",
    threedol: "online",   // ← 新增（基于哪个皮肤）
};
```
> 可用值对应 `ui/styles/character/{skin}.css`、`ui/styles/lbtn/{skin}.css`、`ui/styles/skill/{skin}.css`

**2c.** 加入 `STYLE_TO_INDEX` 映射（决定 player 列数布局）：
```js
const STYLE_TO_INDEX = {
    on: 2,
    off: 1,
    othersOff: 3,
    onlineUI: 4,
    babysha: 5,
    codename: 6,
    threedol: 4,   // ← 新增（等于目标布局的列数）
};
```

### 3. 注册布局标记

**文件**: `src/config/handlers/appearance-handlers.js`

在 `onNewDecadeStyleUpdate` 函数的 `decadeLayoutStyles` 数组中添加：
```js
const decadeLayoutStyles = ["on", "othersOff", "onlineUI", "babysha", "codename", "threedol"];
```
> 在这个数组中 → `dataset.decadeLayout = "on"`（启用十周年布局）

### 4. 注册快捷键

**文件**: `src/features/styleHotkeys.js`

在 `STYLES` 数组中添加：
```js
const STYLES = ["on", "off", "othersOff", "onlineUI", "babysha", "codename", "threedol"];
```
> 决定 alt+数字 快捷键能切换到哪些样式

### 5. 创建样式 CSS

**文件**: `src/styles/player{N}.css`

其中 N = `STYLE_OPTIONS` 中的索引 + 1。

建议从一个已有样式复制作为起点：
```bash
cp src/styles/player4.css src/styles/player7.css   # 以 onlineUI 为基础
```

### 6. 卡牌移动路由（可选）

如果新样式需要自定义的卡牌移动逻辑：

**文件**: `src/overrides/player/card-movement-{style}.js`

创建后无需额外配置，`player/index.js` 会自动通过 `game.checkFile` 检测并加载。

## 修改已有样式

### 修改 player 布局 CSS

直接编辑 `src/styles/player{N}.css`

### 修改 UI 皮肤

对应的皮肤 CSS 文件在 `ui/styles/` 下：
- `ui/styles/character/{skin}.css` — 角色框体
- `ui/styles/lbtn/{skin}.css` — 左侧按钮
- `ui/styles/skill/{skin}.css` — 技能按钮

### 修改卡牌移动逻辑

- 默认实现: `src/overrides/player/card-movement.js`
- 样式专用: `src/overrides/player/card-movement-{style}.js`
